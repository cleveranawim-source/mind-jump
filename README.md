# 🍄 마음 점프! (Mind Jump)

사회정서학습(SEL)을 주제로 한 마리오 스타일 2D 플랫포머 게임입니다.
감정 보석을 모으고, 길에서 만난 동물 친구들의 마음 고민을 함께 풀어 주며 깃발까지 도착하세요!

**▶ 플레이하기: https://cleveranawim-source.github.io/mind-jump/**

## 스테이지 구성 (SEL 4영역)

| 스테이지 | SEL 영역 | 배우는 것 |
|---|---|---|
| 1단계 · 감정 알아차리기 | 자기 인식 | 몸의 신호로 내 감정 알아차리기 |
| 2단계 · 친구 마음 공감 | 대인관계 | 공감하는 말 한마디 |
| 3단계 · 화 다스리기 | 마음 건강 | 멈춤 → 숨 고르기 → 말로 표현 |
| 4단계 · 함께하는 공동체 | 공동체 | 먼저 손 내밀기, 서로 포기하지 않기 |

## 조작법

- **← →** 이동 · **스페이스/↑** 점프 · **1·2·3** 대답 고르기 · **R** 다시 시작
- 모바일: 화면 버튼
- 🎵 버튼으로 배경음악 켜기/끄기

## 특징

- 외부 의존성 없는 **단일 HTML 파일** (폰트만 Google Fonts)
- 배경음악·효과음은 WebAudio로 코드 생성 (음원 파일 없음)
- 오답을 골라도 정죄하지 않고 따뜻한 힌트와 함께 재도전
- 장애물에 닿아도 격려 메시지와 함께 리스폰 (탈락 없음)

## 수업에서 수정하기

`index.html`의 `LEVELS` 배열에서 질문·선택지·피드백을 자유롭게 바꿀 수 있습니다.
지도는 문자 기호로 그립니다: `#` 블록 · `g` 보석 · `N` 친구 · `^` 장애물 · `P` 시작 · `F` 깃발
